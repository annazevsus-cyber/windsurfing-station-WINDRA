# 🏄 WindsurfKG - Виндсёрфинг в Кыргызстане

Красивый и современный сайт-визитка про виндсёрфинг на горных озёрах Кыргызстана.

## 📋 Описание

Сайт представляет информацию о виндсёрфинге как экстремальном водном спорте, популярных местах для занятий в Кыргызстане и способах присоединиться к сообществу.

## 🎯 Функциональность

✅ **Современный дизайн** - красивая и привлекательная визуализация с градиентами и анимациями  
✅ **Полная адаптивность** - оптимально работает на мобильных, планшетах и десктопах  
✅ **Плавная навигация** - smooth scrolling между секциями  
✅ **Интерактивные элементы** - анимации при скролле, hover эффекты  
✅ **Форма обратной связи** - контактная форма с валидацией  
✅ **SEO оптимизация** - правильная структура HTML, мета-теги

## 🚀 Быстрый старт

### Локальный запуск

1. Откройте файл `index.html` в браузере:
   - Просто дважды кликните на файл, или
   - Кликните правой кнопкой → "Open with" → выберите браузер

2. Или используйте локальный сервер:
   ```bash
   # На macOS/Linux с Python 3:
   python3 -m http.server 8000
   
   # На Windows с Python 3:
   python -m http.server 8000
   ```
   Затем откройте: `http://localhost:8000`

### Использование Live Server в WebStorm/VS Code

1. Установите расширение "Live Server" (если используете VS Code)
2. Кликните правой кнопкой на `index.html`
3. Выберите "Open with Live Server"

## 🔧 Кастомизация

### Изменить цвета

Отредактируйте `style.css` и найдите следующие переменные:

```css
.logo-text {
    background: linear-gradient(135deg, #0066cc, #00d4ff);
}
```

### Добавить больше мест для катания

В `index.html` найдите секцию `.spots-grid` и добавьте:

```html
<div class="spot-card">
    <div class="spot-image" style="background: linear-gradient(135deg, цвет1 0%, цвет2 100%);"></div>
    <h3>Название озера</h3>
    <p>Описание...</p>
    <div class="spot-info">
        <span>🏆 Уровень: Указание</span>
        <span>📍 Высота: XXX м</span>
    </div>
</div>
```

### Обновить контактную информацию
